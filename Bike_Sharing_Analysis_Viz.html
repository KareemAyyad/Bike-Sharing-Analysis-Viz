Structure of Dataset¶

This dataset includes 183,412 trips, with 16 features to describe them.

The Dataset's Features are:

  • Trip Duration (seconds)
  • Start Time
  • End Time
  • Start Station ID
  • Start Station Name
  • Stat Station Latitude
  • Start Station Longtitude
  • End Station ID
  • End Station Name
  • End Station Latitude
  • End Station Longtitude
  • Bike ID
  • User Type
  • Member Birth Year
  • Member Gender
  • Bike Sharing For All Trip
  • The Start and End Times are in Date-Time format but their data type is string. Should be converted to Pandas Date-Time.
  • A Rider's Birth Year is provided in the Dataset. We can use that to calculate their current age, in 2021.

The main features of interest in this dataset¶

  • What time of day are trips taking place?
  • What locations are the busiest, and does time affect station traffic?
  • What User Profiles (age, user_type, gender) are riding trips?
  • Do different user profiles correspond to different trips? different times? Do they frequent different stations?
  • Is bike sharing more common amongst certain ages?
  • How long is the average trip?
  • Does average trip time differ if user is a subscriber or customer?

Features of the dataset that may support investigation into questions of interest¶

This is an awesome dataset with comprehensive features. All features of this dataset seem to be relevant to answering the questions posed above.

Conclusion¶

  1. Univariate exploration: 90.5% of the users (riders) and Subscribers, with only 9.5% being Customers. 75% of the users are Males. The average user's age is 36. The 75th percentile is age 41, so most of the users are young. The average trip time is 11.74 Minutes.
  1. Bivariate exploration: There is a positive correlation between age and duration of trips. The top 5 most popular stations where rides started are:
  • Market St at 10th St
  • San Francisco Caltrain Station 2 (Townsend St at 4th St)
  • Berry St at 4th St'
  • Montgomery St BART Station (Market St at 2nd St)
  • Powell St BART Station (Market St at 4th St

We also further validate the findings from Univariate Explorations here.

  1. Multivariate exploration: Segmenting users by their features helps gain insight into the dataset. We discovered that this dataset pertains to only one month of the year. Popular areas, logically, have much higher traffic. There is no clear explanation to why Male riders and significantly more than Female riders. We notice however that most female rides are under 200 minutes.

The stations with the most trips are located in San Francisco and connect to public transportations such as Market St, Caltrain, and Berry St, and Montgomery St Bart.
